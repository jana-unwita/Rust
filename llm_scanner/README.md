# LLM Scanner

A command-line tool that scans your system for installed LLM CLI tools and lets you chat with any one of them interactively.

## Requirements

- [Rust](https://rustup.rs/) (1.80 or later)
- At least one of the supported LLM CLI tools installed on your system

## How to Run

### 1. Clone or navigate to the project

```bash
cd llm_scanner
```

### 2. Build and run

```bash
cargo run
```
